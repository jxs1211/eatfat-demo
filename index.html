About the Game

EATFAT is an exciting MMO (Massively Multiplayer Online) game where players compete in a dynamic environment. Powered by a robust Golang websocket server, the game provides real-time interactions and smooth gameplay.

Key Features

- Real-time multiplayer gameplay
- Powered by Golang websocket server
- Smooth player interactions
- Scalable architecture

Clone our GitHub repository to start playing or contribute to the project. Check out the documentation for setup instructions and gameplay guides.
